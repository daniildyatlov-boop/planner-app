# Планер - Telegram Mini App

Минимальный персональный планер с жестовым интерфейсом для Telegram.

## Быстрый запуск

1. **Локальное тестирование:**
   ```bash
   python3 server.py
   ```
   Откройте http://localhost:8000 в браузере

2. **Тестирование в Telegram:**
   - Установите ngrok: `npm install -g ngrok`
   - Запустите туннель: `ngrok http 8000`
   - Используйте HTTPS URL от ngrok в BotFather

## Основные жесты

### Экран "День"
- **Pull down** — добавить новый план
- **Swipe left** — следующий день  
- **Swipe right** — предыдущий день
- **Swipe по плану влево** — показать кнопку удаления
- **Tap по кнопке удаления** — удалить план

### Экран добавления
- **Enter** — сохранить план
- **Swipe down** — отмена и возврат

## Примеры ввода планов

```
Встреча 15:00
Силовая тренировка 18:00 зал  
Купить продукты
Звонок маме завтра утром
Ужин вечером дом
```

Система автоматически распознает:
- **Время**: "15:00", "утром", "вечером"
- **Дату**: "завтра" (по умолчанию сегодня)
- **Комнату**: последнее слово или известные места

## Настройка в Telegram

1. Создайте бота через @BotFather
2. Получите токен бота
3. Установите Menu Button:
   ```
   /setmenubutton
   @your_bot_name
   Планер
   https://your-ngrok-url.ngrok.io
   ```

## Особенности

- **Gesture-first** интерфейс без кнопок
- **Автоматическое сохранение** в localStorage
- **Адаптивная тема** Telegram
- **Тактильная обратная связь** (вибрация)
- **Парсинг естественного языка** для планов

## Разработка

Для отладки и доработки:
1. Откройте DevTools в браузере
2. Эмулируйте мобильное устройство
3. Тестируйте жесты мышью/тачпадом
4. Проверяйте localStorage для данных

Все изменения применяются мгновенно при обновлении страницы.
